# SimplexEvents
Event plugin specially created for CraftLife

## Permissions:
Permission|Description
---|---
simplexevents.chat.admin|Permission to run chat events
simplexevents.quiz.admin|Permission to run quiz event
simplexevents.enquete.admin|Permission to run polls
simplexevents.evento.admin|Permission to run events
simplexevents.evento.kick|Permission to kikar players of the events
simplexevents.evento.camarote|Permission to access the cabin in events
